What it does

Handy day-to-day tools in one window. Show or hide each one; add more by request.

Launcher

One-click open your favorite apps, files, folders, and links.

Calculator

Keypad plus a type-an-expression evaluator.

Clipboard History

Re-paste anything you copied earlier.

Download Emails

Save a local copy of all your Yahoo Mail — emails and attachments.

One clean home screen

A dashboard of tiles plus a menu — you choose what shows where.

Built to a standard

Every feature is an independent plugin, and quality is enforced on every build.

  • Pluggable host — features load lazily, only when opened
  • Logic in testable classes, not in the UI
  • Unit tests (xUnit) + black-box QA automation
  • Static code analysis (.NET analyzers)
  • Code coverage reported per feature
  • One-command quality gate with a PASS/FAIL verdict
  • Full docs: architecture, user manual, requirements
  • Bi-directional requirements traceability matrix
  • Docs/traceability consistency validator in the build
  • Pre-commit hook + CI so drift can't slip in
