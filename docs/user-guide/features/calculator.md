# Using the Calculator

A quick calculator you can click or type into.

## Open it
**Features → Tools → Calculator**, or click its tile on Home.

## Two ways to use it
- **Keypad:** click the digits and operators (`+ − × ÷`, parentheses, `.`), then **=**.
- **Type it:** click in the display, type an expression like `(12 + 5) * 3 / 2`, and press **Enter**.

## Keys
- **C** clears everything. **⌫** deletes the last character. **=** calculates.

## Tips
- Use parentheses for order of operations, e.g. `(2 + 3) * 4`.
- For a decimal answer, include a decimal, e.g. `7.0 / 2` → `3.5`.

## Troubleshooting
- **"Error":** the expression isn't valid (e.g. `2 +`) or you divided by zero. Press **C** and try again.

## Data
None — the calculator doesn't save anything.
