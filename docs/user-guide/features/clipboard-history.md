# Using Clipboard History

Re-paste things you copied earlier — Windows only keeps the most recent copy.

## Open it
**Features → Tools → Clipboard History**, or click its tile on Home.

## How it works
- Keep the feature open. **Anything you copy** (with Ctrl+C) anywhere in Windows is added to the list,
  newest first.
- The list keeps up to **50** recent text items.

## Re-paste an earlier item
1. **Double-click** the item you want (or select it and click **Copy**).
2. It's now on your clipboard — paste it anywhere with **Ctrl+V**.

## Tidy up
- **Delete** removes the selected item.
- **Clear All** empties the list.

## Privacy
The history is kept **only while the app is open** and is **never saved to disk**.

## Data
None saved — this feature is memory-only.
