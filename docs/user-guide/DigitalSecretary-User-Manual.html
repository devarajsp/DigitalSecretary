Welcome

Digital Secretary collects handy day-to-day tools (“features”) in one window. You choose which ones to show, and new tools can be added over time.

The current version includes a Launcher, a Calculator, a Clipboard History, a Download Emails tool, a Download Gmail tool, and a Download Google Drive tool. Everything runs on your PC and works offline (except the download tools, which need the internet).

Install & run

  1. Make sure you have Windows 10 or 11 and the .NET 9 runtime.
  2. Open the app by running DigitalSecretary.exe.
  3. The main window opens on the Home screen, shown below.
Note
The first time you open a feature it loads instantly and is remembered for the rest of your session.

The main window

The home screen is your dashboard — a tile for each feature you’ve chosen to show.

Ways to open a feature

  • Click a tile on the Home dashboard, or
  • Use the menu: Features → Category → Feature.

Return to Home any time with View → Home. See app info under Help → About.

Show / hide features your choice

Decide which features appear on the dashboard and in the menu — independently.

  1. Click View → Configure Features…
  2. For each feature, tick On Dashboard and/or In Menu.
  3. Click OK. Your choices are saved and applied right away.
Tip
Hid everything by accident? You can always reopen this from View → Configure Features… — that menu never disappears.

Launcher

Open your favorite apps, files, folders, and websites in one click.

Add a shortcut

  • App or file: Add App / File → pick it → give it a name → OK.
  • Folder: Add Folder → choose it → name it → OK.
  • Website: Add URL → type the address → name it → OK.

Open, edit, remove

  1. Open: double-click an item (or select it and click Launch).
  2. Edit: select an item → Edit → change its name/target.
  3. Remove: select an item → Remove → confirm Yes.
Tip
Your list is saved automatically and comes back next time you open the app.
If “Could not open…” appears
The file or folder may have moved, or a link is mistyped — use Edit to fix the target.

Calculator

A quick calculator you can click or type into.

Two ways to use it

  • Keypad: click the digits and operators (+ − × ÷, parentheses, .), then =.
  • Type it: click in the display, type something like (12 + 5) * 3 / 2, and press Enter.

C clears everything · ⌫ deletes the last character · = calculates.

Tip
For a decimal answer, include a decimal point — e.g. 7.0 / 2 gives 3.5.
“Error”?
The expression isn’t valid (e.g. 2 +) or you divided by zero. Press C and try again.

Clipboard History

Windows keeps only the latest copied text. This remembers recent items so you can paste something from earlier.

How to use it

  1. Open Clipboard History and keep it running — new copied text is captured automatically.
  2. To paste an earlier item, double-click it (or select it and click Copy) — it goes back on the clipboard, ready to paste with Ctrl+V.
  3. Use Delete to drop one item, or Clear All to empty the list.
Private by design
The history is kept only while the app is open and is never saved to disk.

Download Emails (Yahoo)

Save a local copy of your Yahoo Mail — every folder, message, and attachment — without changing anything on the server.

You need a Yahoo “app password”
Yahoo does not allow your normal password for this. Create one at Yahoo Account → Account Security → Generate app password and paste it into App password.

Download your mail

  1. Enter your Yahoo email and the app password.
  2. Choose a Save to folder (or accept the default) with Browse….
  3. Click Start Download. Watch the progress bar and log; click Cancel any time.

What you get

  • Your Yahoo folder structure, recreated on your PC.
  • Each email saved twice: a readable .txt and a full-fidelity .eml.
  • All attachments saved as-is, including inline images.
It’s a copy — your mailbox is untouched
Folders are opened read-only, so nothing is deleted, moved, or marked as read in Yahoo. Your email address and folder are remembered; your password is never saved.

Download Gmail

Save a local copy of your Gmail — every folder/label, message, and attachment — without changing anything on the server.

You need a Gmail “app password”
Gmail does not allow your normal password here. Turn on 2-Step Verification, then create one at Google Account → Security → App passwords and paste it into App password. Also enable IMAP in Gmail → Settings → Forwarding and POP/IMAP.

Download your mail

  1. Enter your Gmail address and the app password.
  2. Choose a Save to folder (or accept the default) with Browse….
  3. Click Start Download. Watch the progress bar and log; click Cancel any time.

What you get

  • Your Gmail folder/label structure, recreated on your PC.
  • Each email saved twice: a readable .txt and a full-fidelity .eml.
  • All attachments saved as-is, including inline images.
About Gmail labels
In Gmail a message can carry several labels, so the same message may be saved under more than one folder (for example Inbox and [Gmail]/All Mail).
It’s a copy — your mailbox is untouched
Folders are opened read-only. Your email address and folder are remembered; your password is never saved.

Download Google Drive

Save a local copy of your Google Drive — folders, files, and Google documents — without changing anything in your Drive.

One-time Google setup (OAuth)
Drive has no app password. In the Google Cloud Console, enable the Google Drive API, create an OAuth client ID of type Desktop app, and download its JSON. The first run opens a browser to grant read-only access; the sign-in is then remembered.

Download your Drive

  1. Browse… to your Credentials (.json) file.
  2. Choose a Save to folder (or accept the default).
  3. Click Start Download, approve access in the browser if asked, and watch the progress.

What you get

  • Your Drive folder structure, recreated on your PC.
  • Regular files (PDFs, images, Office files…) downloaded exactly as they are.
  • Google documents exported twice: Docs → .docx + .pdf, Sheets → .xlsx + .pdf, Slides → .pptx + .pdf, Drawings → .png + .pdf.
It’s a copy — your Drive is untouched
The app asks only for read-only access. Your credentials path and folder are remembered; your sign-in token is stored securely in the feature’s own data folder, never in the settings file.

Your data & privacy

Everything stays on your PC under %APPDATA%\DigitalSecretary\:

LocationWhat it holds

data\<feature>\…Each feature’s own data (e.g. your launcher shortcuts).
Privacy
Nothing is uploaded anywhere. Passwords are never stored. Clipboard history lives only in memory.

Troubleshooting / FAQ

A feature showed an error or didn’t open

The rest of the app keeps working. Reopen the feature from the menu. If it keeps happening, note the message shown.

I don’t see a tool I expected

It may be hidden — open View → Configure Features… and tick it for the dashboard and/or menu.

Is my data private?

Yes — everything is local to your PC, and passwords are never saved.
